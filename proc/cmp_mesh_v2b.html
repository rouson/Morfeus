$Id: cmp_mesh_v2b.f90 8157 2014-10-09 13:02:44Z sfilippo $

Description: Sets up vertex to boundary connectivity, so that given a boundary, the GET_ITH_CONN method can be used to obtain the list of vertices that belong to the boundary. Boundary ID's correspond to those used to flag faces:

---------------------------| Vertex Flag | Vertex Type | ---------------------------| 0 | interior | < 0 | proc. bndry | > 0 | phys. bndry | ---------------------------|
